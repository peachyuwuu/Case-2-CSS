Här är en lista på de olika kommandon som används vid versionshantering med git

Vad man behöver göra först:

  1. Skapa en lokal mapp för ditt repository
  2. Skapa en index.html fil i repot i VS Code (touch index.html)
  3. Skapa ett repository på GitHub
  4. Byt till repot i terminalen med cd/ och navigera till repot
  5. Skriv pwd i terminalen för att bekräfta att rätt mapp är vald
  6. Skriv git init för att omvandla mappen till en git mapp
  7. Länka till repot i GitHub med kommandot git remote add origin https://github.com/username/example.git

När man är klar med sin set-up och ska pusha:

  1. git add [file-name]
  2. git commit -m "[changes made]"
  3. git push -u origin main
